# STACK API

## Installation
```bash
    npm install
```

or
``` bash
    yarn add
```

## Config enviroment variables

- Create .env file in the root directory

``` bash
PORT = your_port
SECRET_ACCESS_TOKEN = your_access_token
SECRET_REFRESH_TOKEN = your_refresh_token
```

## Development mode

```bash
    npm run dev
```

or

```bash
    yarn dev
```
